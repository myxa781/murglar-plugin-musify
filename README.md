# Murglar Plugin — Musify

Плагин для [musify.club](https://musify.club) — каталог музыки без регистрации: треки, альбомы, исполнители, жанры, плейлисты.

## Возможности

| Раздел | Описание |
|--------|----------|
| **Топ треков** | /hits — самые популярные треки |
| **Топ релизов** | /top — новые и популярные альбомы |
| **Альбомы** | Полный каталог студийных альбомов с пагинацией |
| **Сборники** | Compilations |
| **Саундтреки** | Soundtracks |
| **Исполнители** | Полный каталог, страница артиста → Треки + Дискография |
| **Жанры** | Все жанры → Треки жанра |
| **Плейлисты** | Тематические плейлисты musify.club |
| **Поиск треков** | Полнотекстовый поиск |
| **Поиск исполнителей** | |
| **Поиск альбомов** | |

## Технические детали

- Авторизация **не требуется**
- Стримминг: прямые MP3-ссылки `/track/dl/{id}/{slug}.mp3`
- Качество: 128 или 320 кбит/с (зависит от трека)
- Парсинг: HTML scraping (публичного API нет)
- API совместимость: Murglar plugin API v7

## Сборка

```bash
cp local.template.properties local.properties
# отредактировать local.properties (пути к keystore)
./gradlew :musify-android:assembleRelease
```

Готовый APK: `musify-android/build/outputs/apk/release/musify-android-release.apk`
